Add Reason

Description: The /add_reason command adds a new reason.

Arguments: short_reason, points, long_reason

Adds a reason to the bot that will be used to punish users. This cab be done by admins only.

View Reasons

Description: The /view_reasons command lists all reasons.

Shows all the reasons in the server, along with the respective points and number of uses.

Edit Reason

Description: The /edit_reason command edits an existing reason.

This can be run by admins only. Opens a form where the punishment data can be submitted, which is then used to punish users.

Delete Reason

Description: The /delete_reason command deletes a reason.

A dropdown is provided from which only admins can delete the reason if they find the need to.
